# 公网访问与部署

## 本机局域网访问

```powershell
$env:HOST="0.0.0.0"
$env:PORT="4173"
npm start
```

同一 Wi-Fi 下的设备访问：

```text
http://你的电脑局域网IP:4173
```

## 公网临时访问

推荐使用 Cloudflare Quick Tunnel：

```powershell
cloudflared tunnel --url http://localhost:4173
```

如果本机没有 `cloudflared`，安装 Cloudflare Tunnel 客户端后再运行上面的命令。也可以把静态文件上传到 Cloudflare Pages、Netlify 或 Vercel。

## 静态托管说明

静态托管时上传：

```text
index.html
assets/
销售文案.md
使用说明.md
定价与复购方案.md
企业微信SOP.md
数据采集报告.md
```

静态模式下没有 `/api/lead`，表单会自动退回浏览器本地缓存，并生成可复制的企业微信话术。
