# 使用说明

## 公网获客页

长期公网地址：

```text
https://a909635137-lgtm.github.io/makemoney/
```

这个页面用于发给闲鱼/小红书意向服务商，展示免费样例和29.9信息包说明。

## 本地销售系统

启动：

```powershell
npm start
```

打开：

```text
http://localhost:4173/sales-console.html
```

本地控制台用于录入潜在客户、生成销售草稿、更新状态、记录订单和推送企微内部提醒。

## 企微机器人

复制 `.env.example` 为 `.env`，填入企业微信群机器人 webhook：

```text
WECOM_BOT_WEBHOOK=你的webhook
```

未配置 webhook 时，系统仍会保存CRM并生成可复制草稿。

## 29.9信息包

生成本周可售ZIP包：

```powershell
npm run build:pack
```

文件会输出到：

```text
dist/packs/
```

## 每周更新

本地一键更新：

```powershell
npm run weekly:update
```

GitHub Actions 也已配置每周一自动采集公开公告并同步公网数据。
