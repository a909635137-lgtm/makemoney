# 使用说明

## 启动

在本目录运行：

```powershell
$env:HOST="0.0.0.0"
$env:PORT="4173"
npm start
```

本机打开 `http://localhost:4173` 或 `http://127.0.0.1:4173`。同一 Wi-Fi 下其他设备打开 `http://你的电脑局域网IP:4173`。

## 怎么卖

1. 让服务商输入服务类型、城市、预算和截止窗口。
2. 网站展示真实公开公告中的 Top 5 跟进机会，并可复制 3 条免费样例。
3. 对方留下微信/手机号后，系统写入 `data/crm/leads_crm.json` 和 `data/crm/leads_crm.csv`。
4. 页面生成企业微信人工确认话术，你复制到企微发送。
5. 样例有效后，卖 `29.9` 完整版、`99/月` 周更或 `299` 定制筛选。

## 当前数据

当前 `assets/data/leads.json` 已内置 30 条真实公开招标/采购公告线索，不含占位 ID。每条包含项目名、采购人/代理机构、地区、预算、公告日期、截止日期、联系人路径、来源 URL、适合服务商、跟进话术和风险提示。

数据来源和抽查记录见 `数据采集报告.md`。后续更新数据时运行：

```powershell
npm run fetch:leads
```

每条线索必须保留具体公告来源链接，不得使用公告首页冒充详情页，不得写确定中标、非公开关系、稳定收益等高风险承诺。
