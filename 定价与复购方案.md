# 定价与复购方案

## 产品阶梯

- 免费样例：3条真实公开公告线索，换取联系方式和服务方向。
- 29.9首包：本周网站/软件信息包，含HTML、CSV、JSON、跟进话术和风险提示。
- 99月更：每周更新一次网站/软件方向公开采购机会。
- 299定制包：指定城市、行业、预算区间，单独筛选并补话术。

## 29.9交付内容

运行：

```powershell
npm run build:pack
```

输出：

```text
dist/packs/服务商订单雷达_网站软件_YYYY-WW_29.9.zip
```

ZIP内含：

- `index.html`：离线可打开的信息包页面。
- `leads.csv`：Excel可打开的线索表。
- `leads.json`：结构化原始数据。
- `README.md`：使用说明。
- `跟进话术.md`：逐条电话和企微话术。
- `风险提示.md`：公开公告核验边界。

## 验证线

- 100个精准网站/软件服务商触达。
- 至少15人愿意看样例。
- 至少3人购买29.9首包。
- 达不到验证线时，不投广告，先改客群、样例质量和首句。
